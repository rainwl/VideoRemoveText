# 视频字幕去除器 — 使用说明

## 第一次使用

1. **解压**：把收到的 `VideoRemoveText.zip` 解压到任意位置（比如桌面）。
2. **打开文件夹**，找到里面的 `start.command` 文件。
3. **右键点击** `start.command`，选择 **打开**（一定要右键，第一次双击 macOS 会拦截）。
   - 如果系统弹窗说"无法验证开发者"，点 **打开** 即可。
4. 会弹出一个**终端窗口**开始安装。**第一次大约需要 3–5 分钟**，会下载：
   - Python 运行时
   - 所需的依赖库
   - AI 模型权重（约 200 MB）
5. 安装完成后浏览器会**自动打开**，地址是 `http://127.0.0.1:7860`。
6. 看到网页就可以开始使用了。

## 使用方法

1. 把视频拖到 **"上传视频"** 区域。
2. 上传后会显示视频第一帧，**红色框**就是要去除字幕的区域。
   - 默认是底部 18%，对大多数视频已经够用。
   - 想调整就拖动下面的 4 个滑块（X / Y / 宽度 / 高度）。
3. 选择**字幕颜色**：白字选 `white`，黑字选 `black`。
4. **建议先勾选"仅预览前 3 秒"**，点击"开始处理"先看看效果。
5. 效果满意后取消勾选，再点一次"开始处理"，等待完整视频处理完成。
6. 处理好的视频会显示在右下角，**点视频右上角的下载按钮**保存。

## 之后再次使用

直接**双击** `start.command` 即可（第二次开始不用右键，也不用再下载东西）。
启动只要几秒钟，浏览器自动打开。

## 退出程序

直接**关掉终端窗口**就行。

## 常见问题

| 问题 | 解决方法 |
|------|---------|
| 双击没反应 / 提示"无法打开" | 右键点 `start.command` → 选 **打开** |
| 终端提示 `command not found: uv` | 关掉终端窗口，重新右键打开 `start.command`（uv 装完后 PATH 需要新窗口才生效） |
| 浏览器没自动打开 | 手动打开浏览器，访问 `http://127.0.0.1:7860` |
| 字幕没去干净 | 在"高级设置"里把"膨胀迭代次数"调到 3-4，或者把"LaMa 额外膨胀"调到 6-8 |
| 输出视频有抖动闪烁 | 在"高级设置"里把"时域平滑窗口"调到 3-4 |
| 处理速度很慢 | 这是正常的，AI 推理本来就慢。建议先用"仅预览前 3 秒"测试 |

## 隐私说明

所有处理都在你自己的电脑上完成，**视频不会上传到任何服务器**。
